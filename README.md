### BEACULT
Application to receive notifications about cultural events when user passes near strategic points with beacons

### Requirements
- Ionic/Angular ^4.7.1
- Angularfire2 ^5.2.1
- Firebase ^7.2.1

### Functionalities
- Signup/Login integrated with Firebase
